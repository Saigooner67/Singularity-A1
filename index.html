Exploring Earth from Space

A School Project on Satellite Data and Environmental Observation

About This Project

This website is part of my school computer project about how satellites help us study our planet. Using satellite imagery and physics principles, scientists can measure things like vegetation health, water levels, and land changes over time.

My goal with this project is to learn how space technology and computer science work together to observe and protect Earth.
